# cellpose-astra

Cellpose 4.0.8 (frozen) with ASTRA-specific training modifications.

This repository exists solely to provide a deterministic, version-locked
Cellpose backend for ASTRA (Automated Structural Tissue Research & Analysis).

Upstream base:
Cellpose v4.0.8

Scope:
- No upstream tracking.
- No general-purpose feature development.
- Modifications limited to ASTRA training integration.

For full framework documentation, see:
ASTRA — Automated Structural Tissue Research & Analysis
